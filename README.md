# bread
Unified framework to stream binary in blocks. (Block Raw Encoding And Decoding)

## Supported codecs
* `qb16` (1:2)
* `qb32` (5:8)
* `qb36` (9.0474:14)
* `qb64` (3:4)
* `qb85` (4.0059:5)
* `qb94` (9.0126:11)
* `qb95` (9.0336:11)
* `qb128` (7:8)
* `korg87` (7:8)
